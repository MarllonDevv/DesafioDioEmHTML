Aprendendo HTML na pratica

1. O que é HTML?

  • Linguagem de marcação usada para criar páginas da web.
  • Estrutura o conteúdo da web com elementos e tags.

2. Estrutura Básica de um Documento HTML

  • < !DOCTYPE html > Declaração do tipo de documento.
  • < html >: Elemento raiz de um documento HTML.
  • < head >: Contém metadados, links para CSS, scripts, etc.
  • < title >: Define o título da página (aparece na aba do navegador).
  • < body >: Contém o conteúdo visível da página (textos, imagens, links, etc).

3. Elementos Principais do HTML

  • Tags de Título: <h1> a <h6 > - Usadas para títulos, onde <h1> é o mais importante.
  • Parágrafos: < p > - Define blocos de texto.
  • Links: < a href="URL" > - Cria hyperlinks.
  • Imagens: < img src="URL" alt="descrição" > - Insere imagens na página.
  • Essas são listas
    • Não ordenadas: < ul > e < li > para criar listas com marcadores.
    1. Ordenadas: < ol > e < li > para listas numeradas.
  • Tabelas: < table >, < tr >, < th >, < td > - Define tabelas.

4. Atributos Comuns das Tags

  • href Para links em < a >.
  • src Para imagens em .
  • alt: Texto alternativo para imagens.
  • id e class: Para identificar elementos de forma única (id) ou por grupo (class), especialmente útil para CSS e JavaScript.

5. Formulários

  • < form >: Define um formulário.
  • Elementos de formulário:
    • < input > Campos de texto, botões de envio, checkboxes, radio
    • < textarea >: Área de texto para várias linhas.
    • < button >: Botão de ação.
    • Atributos Importantes:
      • type Define o tipo de input (text, password, submit etc.).
      • name: Nome do campo.
      • value: Valor padrão do campo.

  • Novas tags do HTML5: Melhoram a organização e acessibilidade da página.

    • : Área de navegação.
    • : Seções de conteúdo.
    • : Artigos independentes.
    • : Rodapé da página.
    • : Conteúdo relacionado lateralmente.

7. Links Internos e Externos

  • Links para páginas internas: href="pagina.html".
  • Links externos: href="http://exemplo.com".

8. Comentários

  • < !-- Comentário -- > Comentários no código que não aparecem no navegador.

9. HTML e CSS

  • HTML define a estrutura; CSS (Cascading Style Sheets) é usado para estilizar o conteúdo.
  • Tag de linkagem: < link rel="stylesheet" href="styles.css" > para vincular arquivos CSS.

10. HTML e JavaScript

  • JavaScript pode ser inserido diretamente em HTML com a tag < script > ou referenciado com um arquivo externo: < script src="script.js" >< /script >.

11. Tags de Mídia

  • Áudio: < audio controls src="arquivo.mp3"> < /audio > - Para incluir áudio com controles.
  • Vídeo: < video controls src="video.mp4" >< /video > - Para vídeos com controles.

12. Boas Práticas

  • Usar semântica apropriada para melhorar acessibilidade e SEO.
  • Fechar todas as tags.
  • Organizar o código para facilitar a manutenção (identação).
